# F# Minesweeper
This is a very minimal implementation of minesweeper in F#. This is also my very first personal coding project, I wanted to see how I would fare in creating a project with a general idea of what I wanted, in contrast to the near step-by-step process of coding my assignments.

It contains only one minesweeper.fsx which contains a minimal and simplistic version of the classic Minesweeper game. The game runs through the terminal.

## How to run
Navigate to it in the terminal and just run the command 'dotnet fsi minesweeper.fsx'.

Commands: r for reveal, f for flag, and x y for coordinates.

# What I have learned
This project wasn't all too difficult since I spent the majority of my first semester coding in F#, however a couple of things really tripped me up at first, and those were system specific. I chose Minesweeper specifically because it is my replacement game for doomscrolling and I have developed an adoration for the game. 

Though I haven't coded much, I prefer functional programming over OOP, and I adore the minimalism and elegance that F# has. It was especially fun to implement patternmatching and pipelines. I have learned, both from the project and from my time in uni, that functional programming comes more intuitively to me than OOP.
